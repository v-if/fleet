# 2. 문제

## 법인차는 늘 수기·분절 관리다

### 관리자
- 위치·상태를 운전자에게 물어봄
- 앱·전화·메신저·엑셀을 오감
- 운행일지·카드·하이패스 3종 대조

### 운전자
- 예약 없이 차 없으면 발동동
- 경고등은 있는데 조치 경로 없음
- 운행일지 작성이 퇴근 숙제

### 대표
- 실시간 운영 현황이 안 보임
- 비용·효율 데이터가 분산
- 도입·축소 판단 근거 부족

월말 = 취합 지옥 · 평일 = 가시성 제로
