# 2. 문제

## 법인차는 아직도 수기·분절 관리다

### 관리자

- 차량 위치·상태를 운전자에게 직접 확인
- 앱·전화·메신저·엑셀을 오가며 관리
- 운행일지·카드·하이패스 내역을 반복 대조

### 운전자

- 예약 정보가 분산되어 차량 배정이 지연
- 차량 이상 발생 시 조치 담당자 확인이 어려움
- 운행일지 작성이 반복 업무

### 대표

- 실시간 운영 현황을 한눈에 보기 어려움
- 비용·운행 효율 데이터가 분산
- 차량 운영 의사결정 근거 부족

월말 = 취합 지옥 · 평일 = 실시간 가시성 부족

# 4. Why Now

- Fleet Telemetry로 실시간 데이터 수집 가능
